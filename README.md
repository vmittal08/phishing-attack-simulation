# 🔐 Phishing Attack Simulation and Detection

## 📌 Objective
This project demonstrates how phishing attacks work and implements basic detection mechanisms to identify suspicious login attempts.

## 🛠️ Tools Used
- Python
- Flask

## ⚙️ Features
- Fake login page (Gmail-style)
- Captures user credentials (simulation only)
- Stores data in a file
- Detects suspicious inputs
- Displays warning messages

## 🚀 How to Run

1. Install Flask:
   pip install flask

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000

## 🧪 Detection Logic
- Checks if email contains '@'
- Checks password length
- Flags suspicious inputs


## ⚠️ Disclaimer
This project is created for educational purposes only to understand phishing attacks and improve cybersecurity awareness.
